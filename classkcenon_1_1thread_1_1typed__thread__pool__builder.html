Detailed Description

template<detail::JobType job_type = job_types>
class kcenon::thread::typed_thread_pool_builder< job_type >

Typed thread pool builder template.

Builder class for constructing typed thread pools.

This class provides a fluent interface for building typed thread pools with various configuration options. It ensures consistent construction and validation of pool parameters.
