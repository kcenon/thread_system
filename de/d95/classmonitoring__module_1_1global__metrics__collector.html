Detailed Description

Singleton instance for application-wide metrics collection.

Provides a convenient global access point for metrics collection functionality. Useful for applications that need a single, shared metrics collector accessible from multiple components.

Thread Safety

All methods are thread-safe. The singleton instance is lazily initialized with thread-safe guarantees.

Example

// Initialize global collector
monitoring_config config;
config.collection_interval = std::chrono::milliseconds(50);
auto& global = global_metrics_collector::instance();
if (auto result = global.initialize(config); !result) {
// Handle error
}
// Use from anywhere in the application
auto collector = global.get_collector();
auto metrics = collector->get_current_snapshot();
