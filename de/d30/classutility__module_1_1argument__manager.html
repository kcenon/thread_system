Detailed Description

Manages command line arguments and provides type-safe retrieval of values.

This class can parse command line arguments supplied either as a single string or via argc/argv. It stores arguments in a key-value map (e.g., "--key" : "value") and provides functions to retrieve these values as various data types (string, bool, int, etc.).
