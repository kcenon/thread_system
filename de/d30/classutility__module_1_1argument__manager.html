Detailed Description

A utility class for parsing and managing command-line arguments.

The argument_manager class provides functions to:

  • Parse command-line arguments from either a single string (e.g., "--option value") or the traditional argc, argv approach.
  • Store parsed arguments in a key-value map of type std::map<std::string, std::string>.
  • Retrieve values by key in a type-safe manner (e.g., bool, int).

Parsing Conventions

  • Arguments are expected in the form: --key value.
  • If an argument has no following value, it may be stored with an empty string ("").
  • If a key appears multiple times, the last occurrence typically overrides previous ones.

Example Usage

int main(int argc, char* argv[])
{
utility_module::argument_manager arg_mgr;
if (auto err = arg_mgr.try_parse(argc, argv)) {
std::cerr << "Error parsing arguments: " << *err << std::endl;
return EXIT_FAILURE;
}
// Retrieve a string option
if (auto username = arg_mgr.to_string("--username")) {
std::cout << "Username: " << *username << std::endl;
}
// Retrieve a bool option
if (auto debug_mode = arg_mgr.to_bool("--debug")) {
std::cout << "Debug mode: " << (*debug_mode ? "on" : "off") << std::endl;
}
return 0;
}

Parse command-line arguments from a single (possibly wide) string.

Template Parameters
StringTypeThe type of the input string (e.g., std::string or std::wstring).
Parameters
argumentsThe argument string, which may look like --option1 value1 --flag --option2 value2.
Returns
std::optional<std::string> containing an error message on failure, or std::nullopt on success.

Internally, this method:

  1. Converts the input string to a UTF-8 or system-encoded narrow string if necessary.
  2. Splits the string into tokens based on whitespace.
  3. Calls parse() to populate arguments_ with key-value pairs.

Example

argument_manager mgr;
std::string cmd = "--user alice --debug --count 10";
if (auto err = mgr.try_parse(cmd)) {
std::cerr << "Parse error: " << *err << std::endl;
}
