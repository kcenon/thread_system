enum class thread_module::thread_conditions : uint8_t
strong

Enumeration of job priority levels.

Defines different levels of job priorities for the thread pool. This enum class is based on uint8_t for efficient storage.

Enumerator
Created 

Thread created but not started.

Waiting 

Thread waiting for a job.

Working 

Thread working a job.

Stopping 

Thread stopping.

Stopped 

Thread stopped.
