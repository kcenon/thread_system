Detailed Description

Performance metrics for individual worker threads.

Tracks per-worker statistics including job processing count, processing time, idle time, and context switches. Useful for identifying worker load imbalances or performance issues.

Thread Safety

All member variables are atomic for lock-free concurrent access.
