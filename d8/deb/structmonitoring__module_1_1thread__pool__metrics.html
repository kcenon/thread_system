Detailed Description

Performance metrics for thread pool operations.

Tracks job processing statistics, queue depth, and worker thread utilization. Provides insights into thread pool efficiency and throughput.

Thread Safety

All member variables are atomic, ensuring thread-safe concurrent access without explicit synchronization.
