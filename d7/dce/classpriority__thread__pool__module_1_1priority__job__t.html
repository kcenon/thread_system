Detailed Description

template<typename priority_type>
class priority_thread_pool_module::priority_job_t< priority_type >

Represents a job that carries a specific priority level.

This class extends the base job interface to include a priority value that is used by a priority-based scheduling system. The class also keeps a weak reference to a priority_job_queue_t, which manages this job. By using a weak pointer, the queue avoids circular references that could prevent proper resource cleanup.

Template Parameters
priority_typeThe data type used to represent the priority level. Typically, an enum or an integral type.
