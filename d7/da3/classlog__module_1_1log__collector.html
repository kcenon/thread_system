Detailed Description

A class for collecting and managing log messages.

This class inherits from thread_base and provides functionality for collecting, processing, and distributing log messages to console, file, and callback outputs. It manages separate queues for log collection and output handling. The class supports various string types for log messages including std::string and std::wstring.
