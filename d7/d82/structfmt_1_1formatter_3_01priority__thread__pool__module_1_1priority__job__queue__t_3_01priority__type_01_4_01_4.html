Detailed Description

template<typename priority_type>
struct fmt::formatter< priority_thread_pool_module::priority_job_queue_t< priority_type > >

Specialization of fmt::formatter for priority_job_queue_t<priority_type> when using the {fmt} library.

This allows a priority_job_queue_t<priority_type> to be formatted using the {fmt} library, by invoking the to_string() method on the instance.
