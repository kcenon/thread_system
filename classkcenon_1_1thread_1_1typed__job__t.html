Detailed Description

template<typename job_type>
class kcenon::thread::typed_job_t< job_type >

Typed job template.

Represents a job that carries a specific priority level.

This class extends the base job interface to include a priority value that is used by a priority-based scheduling system. The job uses the base class's weak reference to a job_queue for queue management.

Template Parameters
job_typeThe data type used to represent the priority level. Typically, an enum or an integral type.
