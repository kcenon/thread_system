Detailed Description

Public forwarding header for type-safe thread pool.

Provides a stable include path for typed_thread_pool, which offers compile-time type safety for job submission and result retrieval.

#include <kcenon/thread/core/typed_thread_pool.h>
// Create a typed thread pool for string processing
typed_thread_pool<std::string> pool("string_pool", 2);
// Submit typed work
pool.enqueue([](const std::string& input) {
return "processed: " + input;
}, "hello");
