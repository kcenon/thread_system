Detailed Description

template<typename T>
class thread_pool_module::task_promise< T >

Promise type for coroutine task implementation.

This class implements the promise type requirement for C++20 coroutines using the task<T> return type. It provides the necessary methods to create and manage coroutines, handle return values, and propagate exceptions.

Template Parameters
TThe type of value that will be produced by the coroutine
