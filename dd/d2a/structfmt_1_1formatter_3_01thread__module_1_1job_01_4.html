Detailed Description

Specialization of fmt::formatter for thread_module::job.

Allows objects of type job to be formatted using the {fmt} library, converting them to a string via job::to_string() internally.

Example

auto my_job_ptr = std::make_shared<thread_module::job>("example_job");
std::string output = fmt::format("Job info: {}", *my_job_ptr); // calls to_string()
