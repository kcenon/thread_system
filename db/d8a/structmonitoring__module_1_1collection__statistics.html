Detailed Description

Statistics about the metrics collection process.

Tracks the performance and health of the metrics collector itself, including collection frequency, errors, and buffer overflows. Useful for monitoring the monitoring system.

Thread Safety

All members are atomic, allowing concurrent access without locks.
