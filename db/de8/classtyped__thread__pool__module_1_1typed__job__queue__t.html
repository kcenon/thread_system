Detailed Description

template<typename job_type = job_types>
class typed_thread_pool_module::typed_job_queue_t< job_type >

A template-based queue that manages jobs with distinct priority levels.

This class inherits from job_queue and provides functionality to enqueue and dequeue priority-based jobs. Internally, it maintains multiple queues (one per priority level) and tracks their sizes.

Template Parameters
job_typeThe type used to represent job priority levels. Defaults to job_types.
