Detailed Description

Implementation of a coroutine-based task system.

This file provides the implementation of a C++20 coroutine-based asynchronous task system. It includes the necessary promise types, awaitable objects, and task classes to enable writing asynchronous code with the co_await syntax. The system supports error handling through the result<T> and result_void types.
