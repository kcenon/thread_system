Detailed Description

Cross-platform CPU affinity management for threads.

This class provides a unified interface for setting and querying CPU affinity across Windows, Linux, and macOS platforms. CPU affinity allows binding threads to specific CPU cores, which can improve performance by:

  • Reducing cache misses through better locality
  • Minimizing thread migration overhead
  • Providing more predictable performance characteristics
Note
macOS has limited support for CPU affinity compared to Windows and Linux
