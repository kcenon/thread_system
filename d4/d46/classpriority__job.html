Detailed Description

Represents a job with a priority level.

This template class encapsulates a job with a callback function, a priority level, and a name. It inherits from job and std::enable_shared_from_this to allow creating shared_ptr from this.

Template Parameters
priority_typeThe type used to represent the priority level.
