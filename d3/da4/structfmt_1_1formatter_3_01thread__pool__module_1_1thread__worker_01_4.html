Detailed Description

Specialization of fmt::formatter for thread_pool_module::thread_worker.

Allows thread_worker objects to be formatted as strings using the {fmt} library.

Example

auto worker = std::make_unique<thread_pool_module::thread_worker>();
std::string output = fmt::format("Worker status: {}", *worker);
