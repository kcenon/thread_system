Detailed Description

Specialization of fmt::formatter for thread_pool_module::thread_pool.

Allows thread_pool objects to be formatted as strings using the {fmt} library.

Example

auto pool = std::make_shared<thread_pool_module::thread_pool>("MyPool");
pool->start();
std::string output = fmt::format("Pool Info: {}", *pool);
