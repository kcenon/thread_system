Submit a callable with configurable options.

Submit a callable with configurable options.This is the unified API for submitting single tasks. It replaces submit_async with a more flexible options-based approach.

Template Parameters
FCallable type
RReturn type (automatically deduced)
Parameters
callableThe function to execute
optsOptional submit options (name, etc.)
Returns
Future for the result
// Basic usage (equivalent to submit_async)
auto future = pool->submit([]{ return 42; });
int result = future.get();
// With named job
auto future = pool->submit([]{ return 42; }, {.name = "compute"});
// Using static factory
auto future = pool->submit([]{ return 42; }, submit_options::named("task"));
