Detailed Description

template<typename job_type>
class kcenon::thread::aging_typed_job_t< job_type >

A typed job with priority aging support.

This class extends typed_job_t to include aging information, allowing the job's effective priority to increase over time based on how long it has been waiting in the queue.

Template Parameters
job_typeThe type used to represent priority levels.

Features

  • Tracks enqueue time automatically
  • Maintains current priority boost
  • Provides effective priority calculation
  • Supports configurable max boost

Example Usage

auto job = std::make_unique<aging_typed_job_t<job_types>>(
job_types::Background,
[]() -> common::VoidResult {
// Do work
return common::ok();
},
"background_task"
);
// Later, apply boost
job->apply_boost(1);
auto effective = job->get_aged_priority().effective_priority();
