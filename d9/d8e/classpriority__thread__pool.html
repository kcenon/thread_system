Detailed Description

A thread pool class that manages priority-based jobs and workers.

This template class implements a thread pool that manages jobs and workers with different priority levels. It inherits from std::enable_shared_from_this to allow creating shared_ptr from this.

Template Parameters
priority_typeThe type used to represent the priority levels.
