Detailed Description

Specialization of fmt::formatter for thread_module::thread_base.

Enables formatting of thread_base objects as strings using the {fmt} library.

Example

auto worker = std::make_shared<my_thread>();
std::string output = fmt::format("Thread info: {}", *worker);
