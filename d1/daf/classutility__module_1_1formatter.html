Detailed Description

Provides convenience methods for string formatting using either C++20 <format> or {fmt} library.

The formatter class offers static functions to format strings (both narrow and wide) into a std::string / std::wstring, or directly to an output iterator. Which underlying implementation is used depends on whether USE_STD_FORMAT is defined.

Example

// Basic usage:
std::string result = formatter::format("Hello, {}", "World");
// Writing directly to a buffer:
std::array<char, 50> buffer;
auto it = formatter::format_to(buffer.begin(), "Number: {}", 42);
// 'it' now points to the end of the written text
